2.朴素贝叶斯算法

贝叶斯定理是朴素贝叶斯算法的基础，其公式为：

$P(C|X) = \frac{P(X|C) \times P(C)}{P(X)}$

其中：

  • ( P(C|X) ) 是后验概率，即在给定特征 ( X ) 的条件下，样本属于类别 ( C ) 的概率。
  • ( P(X|C) ) 是条件概率，即在类别 ( C ) 下，特征 ( X ) 出现的概率。
  • ( P(C) ) 是先验概率，即类别 ( C ) 出现的概率。
  • ( P(X) ) 是特征 ( X ) 出现的总概率。

在实际应用中，( P(X) ) 是一个常数，因此分类决策主要依赖于 ( P(X|C) ) 和 ( P(C) )。

  1. 决策树算法

决策树通过递归地选择最优特征并将其分割，逐步构建树的结构。每次分裂的目标是使分裂后的子节点更加“纯净”（即属于同一类别的样本更多）。决策树算法容易过拟合。

  1. 逻辑回归算法

说是回归其实这是一种分类算法，广泛应用于二分类问题的统计学习方法。

逻辑回归的核心思想是通过逻辑函数（Sigmoid函数）将线性回归的输出映射到(0, 1)区间，从而实现对样本类别的概率预测。

  1. 集成方法——梯度提升树GBDT

梯度提升决策树是一种基于Boosting思想的集成学习算法，通过迭代训练多个决策树来逐步优化模型的预测性能。其核心思想是将多个弱学习器（通常是决策树）组合成一个强学习器，每棵树都拟合前一轮模型的残差，从而逐步减少误差。

  1. 其他
  • xgboost

GBDT的损失函数支队误差部分做负梯度（一阶泰勒展开）

XGBoost的损失函数对误差部分做二阶泰勒展开，更加准确，更快收敛。

  • linghtgbm

微软：快速的，分布式的，高性能的基于决策树算法的梯度提升框架，速度更快。

  • stacking

堆叠或者叫模型融合
先建立几个简单的模型进行训练，第二级学习器会基于前级模型的预测结果进行再训练。

  • 神经网络

神经网络的训练目标是最小化预测值与真实值之间的误差。
