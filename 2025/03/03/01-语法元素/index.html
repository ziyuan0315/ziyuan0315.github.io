第二部分 变量

1. 变量

  • 量体现在，这是一个实在的对象，或是具体的数据，或是抽象的运算结果
  • 变体现在，可以进行运算、查找、修改、删除等操作
  • 两个要素： 变量名和赋值

2. 变量的命名

2.1 那些可以用来做变量名

  • 大小字母、数字、下划线、汉字机器组合
  • 严格区分大小写

2.2 以下情况不可以做变量名

  • 首字母为数字
  • 变量名中间有空格
  • 与Python保留字相同

第五部分 程序格式

1.行最大长度

所有行限制的最大字符数为79

2.缩进

  • 用缩进来表示语句间的逻辑
  • 在for while def class等：之后下一行开始进行缩进，表示代码与前句之间的从属关系
  • 缩进量:一般用Tab键，4字符

3.使用空格

  • 二元运算符两边加一个空格

  • 使用不同优先级的运算符，考虑在最低优先级的运算符周围添加空格

  • 在逗号后面使用空格

  • 不要使用一个以上的空格

4. 避免使用空格

  • 在制定关键字参数或者默认参数值的时候，不要在=附近加上空格

  1. 以上属于PEP8格式指南的部分内容，养成良好的编码习惯利人利己

    点击了解更多PEP8（中英对照）

  2. 格式约定的目的：

    • 使大量Python代码风格一致
    • 提升可读性

  3. 在实际项目中，项目要求与代码规范相冲突时，项目优先
